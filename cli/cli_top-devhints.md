### Shortcuts

| Shortcut  | Description           |
| --------- | --------------------- |
| `h`       | shows help            |
| `q`       | quits the program     |
| `m`       | switches memory view  |
| `k`       | kills process         |
| `Shift+p` | sorts by CPU usage    |
| `Shift+m` | sorts by memory usage |
| `Shift+r` | reverses sorting      |
| `Shift+l` | searches for string   |
| `o`       | adds a filter         |
| `=`       | clears filters        |
